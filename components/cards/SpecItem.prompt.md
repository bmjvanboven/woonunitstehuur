Icon + label spec line for unit features (surface area, bedrooms, occupancy). Gold-tinted icon chip.

```jsx
<SpecItem icon={<BedIcon/>} value="2" label="slaapkamers" />
<SpecItem icon={<RulerIcon/>} label="ca. 30 m²" />
```

Props: `icon`, `label`, `value` (optional emphasized number), `direction` ('row' | 'column').
