Icon-only button for compact actions (close, navigation arrows, phone). Always pass `label` for accessibility.

```jsx
<IconButton label="Volgende" variant="primary" round>
  <ChevronRight />
</IconButton>
```

Variants match Button: `primary` · `secondary` · `outline` · `ghost`. Sizes `sm`/`md`/`lg`. `round` makes it circular.
