# Website UI kit — Woonunitstehuur.nl

A high-fidelity, interactive recreation of the Woonunitstehuur.nl marketing website, composed from the design system's own components (`Button`, `Badge`, `UnitCard`, `SpecItem`, `Input`, `Select`, `Textarea`).

## Screens
- **HomeScreen** — hero (see Hero.jsx), USP grid, featured units, 4-step "zo werkt het", gold CTA band.
- **Hero** — bright, airy "Kaart" layout: text left, a rounded photo card right (warm furnished interior) with an overlapping "Direct beschikbaar · vanaf € 595" card.
- **AanbodScreen** — units listing with filter chips + sort select.
- **UnitDetailScreen** — gallery + thumbnails, included-features list, sticky request sidebar with specs & price.
- **ContactScreen** — quote-request form (with success state) + dark contact-info panel.

## Run
Open `index.html`. It loads `../../_ds_bundle.js` (generated by the compiler), then the Babel JSX screens. Navigation is fake client-side state via `onNav(key)`.

## Notes / substitutions
- **Imagery:** the kit uses **real Woonunitstehuur project photos** (`assets/hero.jpg`, `unit-comfort/gezin/familie.jpg`, `interior-dining/bedroom.jpg`, `placing.jpg`). `assets/*-placeholder*.png` remain as neutral fallbacks.
- **Icons** are Lucide-style 2px line icons defined in `icons.jsx` (`window.WUIcons`) — the live site has no documented custom icon set.
- Phone number and address are illustrative.
- The visual direction (gold + ink, Archivo/Hanken type) was derived from the brand logo and site copy; the live site's exact theme code was not available.
